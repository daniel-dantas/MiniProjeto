## Instruções de uso

**1** - Mudar as variaveis de ambiente no arquivo **.env** na raiz do projeto

**2** - Instalar as dependências do sistema com:

```
yarn
```
**ou**
```
npm install
```
**3** - Executar o projeto
```
yarn dev
```
**ou**
```
npm run dev
```
